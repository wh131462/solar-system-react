# Solar System React

一个基于 React + Three.js 构建的交互式 3D 太阳系模拟器，集探索、学习、实验于一体。

[**在线体验**](https://wh131462.github.io/solar-system-react/)

## 功能特性

### 自由探索
- 3D 太阳系全景，包含太阳及八大行星与卫星
- 程序化生成的行星纹理（地球大陆与云层、气态巨行星条纹等）
- 轨道轨迹、引力场可视化
- 点击行星聚焦查看，支持键盘快捷键导航

### 知识图谱
- **3D 剖面图**：查看行星内部结构层次
- **百科卡片**：行星信息轮播展示
- **大小对比**：行星体积直观比较
- **时间线**：人类太空探索里程碑

### 物理实验室
- 实时调节轨道质量（0.1x ~ 20x）、半径、离心率
- 可视化观察参数变化对轨道运动的影响

### 今晚观测
- 根据日期和时间，展示当晚可观测的行星
- 显示亮度等级与方位信息

### 更多
- 12 个可解锁成就（太阳系旅行者、天文学博士、深空探索者等）
- 模拟速度控制（0.1 小时/秒 ~ 365 天/秒）
- 真实/演示比例切换
- 每颗行星附带趣味知识、探索历史与互动问答

## 技术栈

| 技术 | 说明 |
|------|------|
| React 19 | UI 框架 |
| TypeScript | 类型安全 |
| Three.js | 3D 渲染引擎 |
| Vite 8 | 构建工具 |

## 快速开始

```bash
# 克隆项目
git clone https://github.com/wh131462/solar-system-react.git
cd solar-system-react

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 暂停 / 播放模拟 |
| `0-9` | 快速跳转到对应行星 |
| `Esc` | 重置视角 |
